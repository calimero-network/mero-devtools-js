Quick Start

1

Install

npm install @calimero/abi-codegen
2

Generate

# From your ABI manifest file:
npx calimero-abi-codegen \
  -i abi.json \
  -o src/generated
3

Use generated client

import { AppClient } from './generated';

const client = new AppClient(nodeUrl);
const result = await client.getValue({key:'foo'});
