ABI Format

JSON schema emitted by the Calimero Rust SDK — the input consumed by mero-devtools

Top-level Structure

// abi.json
{
  "version": "1",
  "methods": [ /* AbiMethod[] */ ],
  "types":   [ /* AbiTypeDef[] */ ]
}
version
Schema version string. Currently "1". Used for forward-compatibility checks.
methods
Array of all public methods exposed by the WASM application.
types
Array of all named types (structs, enums) referenced by method signatures.

AbiMethod

{
  "name": "get_post",           // Rust method name (snake_case)
  "kind": "query",             // "query" | "mutate"
  "params": [
    { "name": "id", "type": "u32" }
  ],
  "returns": { "type": "Post" }  // null if void
}
kind: "query"
Read-only method. Maps to rpc.query() in the generated client.
kind: "mutate"
State-changing method. Maps to rpc.mutate() in the generated client.

AbiTypeDef

Struct

{
  "name": "Post",
  "kind": "struct",
  "fields": [
    { "name": "id", "type": "u32" },
    { "name": "title", "type": "String" }
  ]
}

Enum

{
  "name": "Status",
  "kind": "enum",
  "variants": [
    { "name": "Active" },
    { "name": "Archived" }
  ]
}
