API Reference

Programmatic API for integrating codegen into build tooling

generate(options)

Main entry point when using mero-devtools-js as a library rather than a CLI:

import { generate } from '@calimero-network/mero-devtools-js';

await generate({
  input: './abi.json',       // ABI JSON file path
  output: './src/client.ts',// Output TypeScript path
  className: 'TodoClient', // Generated class name
  rpcImport: '@calimero-network/mero-js',
});

GenerateOptions

input: string
Required. Path to the ABI JSON file.
output: string
Required. Path where the TypeScript file will be written.
className?: string
Name for the generated client class. Default: 'AppClient'.
rpcImport?: string
Module specifier for the RpcClient import. Default: '@calimero-network/mero-js'.
noTypes?: boolean
Skip emitting TypeScript interface/type declarations. Default: false.

parseAbi(json)

Parse and validate an ABI JSON object. Returns an Abi AST or throws on invalid input:

import { parseAbi } from '@calimero-network/mero-devtools-js';

const raw = JSON.parse(fs.readFileSync('abi.json', 'utf-8'));
const abi = parseAbi(raw);

// abi.methods — array of AbiMethod
// abi.types — array of AbiTypeDef (structs / enums)

renderClient(abi, options)

Render a TypeScript string from a parsed ABI. Returns the raw TypeScript source code:

import { parseAbi, renderClient } from '@calimero-network/mero-devtools-js';

const abi = parseAbi(raw);
const ts = renderClient(abi, {
  className: 'TodoClient',
});
fs.writeFileSync('src/client.ts', ts);
